# "Seeed Studio Grove" connector adapter

Enables many of the [Grove expansion modules](https://wiki.seeedstudio.com/Grove_System/) to be used with the badge.

![Top render](/images/renderf.png)

* I2C devices such as sensors
* I/O 1 uses HS_F and HS_G pins
* I/O 2 uses HS_H and HS_I pins

Use of pins is described at [Create a hexpansion](https://tildagon.badge.emfcamp.org/hexpansions/creating-hexpansions/)

# What's included

* kicad - The Kicad project files for the hexpansion along with gerbers ready to send to a PCB fabricator.
* software - Example applications for the hexpansions, using DS18B20 temperature probe and a Grove alcohol gas sensor.
